# Sample ASP.NET MVC Application

Simple web application that stores, edits and displays widgets entered into a database.<br/>
  
**User Actions:**  
- Create  
- Edit  
- Delete  
  
**Data Fields:**  
- Name  
- Time Created  
- Type  
- Sub-Type
